# Northwind

Next.js landing page for Northwind, migrated from the original React + Vite setup.

## Scripts

- `npm run dev` starts the Next.js dev server on port 8080.
- `npm run build` creates a production build.
- `npm run start` serves the production build on port 8080.
- `npm run lint` runs ESLint.
- `npm run test` runs Vitest.
